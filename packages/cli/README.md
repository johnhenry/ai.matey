# ai.matey.cli

CLI tools for AI Matey - proxy server, format converters, Ollama emulation

Part of the [ai.matey](https://github.com/johnhenry/ai.matey) monorepo.

## Installation

```bash
npm install ai.matey.cli
```

## Usage

```bash
npx ai-matey --help
```

## Commands

See `ai-matey --help` for available commands.
