# ai.matey.utils

Shared utility functions for streaming, validation, and more

Part of the [ai.matey](https://github.com/johnhenry/ai.matey) monorepo.

## Installation

```bash
npm install ai.matey.utils
```

## Exports

- `asyncGeneratorToReadableStream`
- `readableStreamToAsyncGenerator`
- `collectStreamChunks`

## Usage

```typescript
import { asyncGeneratorToReadableStream, readableStreamToAsyncGenerator, collectStreamChunks } from 'ai.matey.utils';
```
