# ai.matey.react.hooks

Additional React hooks for AI Matey - useAssistant, useStream, useTokenCount

Part of the [ai.matey](https://github.com/johnhenry/ai.matey) monorepo.

## Installation

```bash
npm install ai.matey.react.hooks
```

## Usage

```typescript
import { /* hooks or components */ } from 'ai.matey.react.hooks';
```

See the [React integration guide](https://github.com/johnhenry/ai.matey/tree/main/docs) for detailed usage.
