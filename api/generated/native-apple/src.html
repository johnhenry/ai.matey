native-apple/src

Apple Backend Adapter

Backend adapter using Apple's Foundation Models via apple-foundation-models. Only works on macOS 15+ (Sequoia) with Apple Intelligence.

Gracefully fails on unsupported platforms (Linux, Windows, older macOS).

Example​

import { AppleBackend } from 'ai.matey.native.node-llamacpp';

const backend = new AppleBackend({
  maxTokens: 2048,
  temperature: 0.7,
});

await backend.initialize();
const response = await backend.execute(request);
