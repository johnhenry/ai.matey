native-model-runner/src

Generic Model Runner Backend

Base class for backends that run models via subprocess/binary execution. Handles process lifecycle, stdio communication, and common functionality.

Classes​

abstract GenericModelRunnerBackend​

Defined in: packages/native-model-runner/src/index.ts:37

Abstract base class for model runner backends.

Handles:

  • Process spawning and lifecycle
  • stdio communication
  • Health checking
  • Event emission
  • Stats tracking

Subclasses must implement:

  • buildCommandArgs() - Convert config to CLI args
  • formatPrompt() - Convert IR request to prompt string
  • parseResponse() - Parse model output to IR response
  • parseStreamChunk() - Parse streaming output to IR chunks
