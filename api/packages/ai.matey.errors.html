ai.matey.errors

Error classes for AI Matey - Universal AI Adapter System

Overview​

This package is part of the ai.matey ecosystem.

Exports​

Values​

ErrorCodeEnum​

ERROR_CODE_CATEGORIES​

AdapterError​

Base error class for all adapter errors.

AuthenticationError​

Authentication error (API key issues).

AuthorizationError​

Authorization error (permissions, quota).

RateLimitError​

Rate limit error with retry information.

ValidationError​

Validation error with field-specific context.

ProviderError​

Provider error (upstream API issues).

AdapterConversionError​

Adapter conversion error (translation issues).

NetworkError​

Network error (connectivity issues).

StreamError​

RouterError​

Router error (routing issues).

MiddlewareError​

Middleware error.
