ai.matey.core

Core components for AI Matey - Universal AI Adapter System

Overview​

This package is part of the ai.matey ecosystem.

Exports​

Values​

Bridge​

Bridge connects frontend and backend adapters.

createBridge​

Create a new Bridge instance.

Router​

Router manages multiple backend adapters with intelligent routing.

createRouter​

Create a new Router instance.

createRouter(config?: Partial<RouterConfig>): Router

MiddlewareStack​

Middleware stack for composing middleware functions.

createMiddlewareContext​

Create middleware context from request and config.

createMiddlewareContext(request: IRChatRequest, config: Record<string, unknown>, signal?: AbortSignal ): MiddlewareContext

Create streaming middleware context from request and config.

Get pricing information for a specific model.

getModelPricing(modelId: string): ModelPricing | null

getModelCapabilities​

Get extended capabilities for a specific model.

getModelCapabilities(modelId: string): Partial<ExtendedModelCapabilities> | null

getAllPricedModels​

Get all models with pricing data.

getAllPricedModels(): string[]

getModelsByFamily​

Find models by model family.

getModelsByFamily(family: string): string[]

setPricingOverride​

Set custom pricing for a model (overrides static database).

setPricingOverride(modelId: string, pricing: ModelPricing): void

clearPricingOverride​

Clear pricing override for a model.

clearPricingOverride(modelId: string): void

clearAllPricingOverrides​

clearAllPricingOverrides(): void

getPricingWithOverrides​

getPricingWithOverrides(modelId: string): ModelPricing | null

matchModels​

Match and score models based on capability requirements.

matchModels(requirements: CapabilityRequirements, availableModels: BackendModel[] ): ScoredModel[]

filterByRequirements​

Filter models by hard requirements only.

filterByRequirements(availableModels: BackendModel[], requirements: CapabilityRequirements['required'] ): BackendModel[]

findBestModel​

Find the best model match for requirements.

findBestModel(requirements: CapabilityRequirements, availableModels: BackendModel[] ): ScoredModel | null

getTopMatches​

Get top N model matches.

getTopMatches(requirements: CapabilityRequirements, availableModels: BackendModel[], count: number = 3 ): ScoredModel[]

inferCapabilities​

inferCapabilities(modelName: string, metadata?: Record<string, unknown> ): Partial<ModelCapabilities>

mergeCapabilities​

Merge multiple capability sources with priority.

mergeCapabilities(...sources: Array<Partial<ModelCapabilities> | null | undefined> ): Partial<ModelCapabilities>

meetsRequirements​

Check if a model meets minimum capability requirements.

meetsRequirements(capabilities: Partial<ModelCapabilities>, requirements: Partial<ModelCapabilities> ): boolean

calculateCapabilitySimilarity​

Calculate similarity score between two capability sets.

calculateCapabilitySimilarity(caps1: Partial<ModelCapabilities>, caps2: Partial<ModelCapabilities> ): number

createModelTranslator(defaultOptions: ModelTranslationOptions ): (modelName: string, overrides?: Partial<ModelTranslationOptions>) => TranslationResult
