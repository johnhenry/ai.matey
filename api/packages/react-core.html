ai.matey.react.core

Core React hooks for AI Matey - useChat, useCompletion, useObject

Overview​

This package is part of the ai.matey ecosystem.

Exports​

Values​

useChat​

useChat - React hook for chat interfaces.

useChat(options: UseChatOptions = {}): UseChatReturn

useCompletion​

useCompletion - React hook for text completion.

useCompletion(options: UseCompletionOptions = {}): UseCompletionReturn

useObject​

useObject - React hook for streaming structured objects.
